   - `notify()` — call each listener with this

4. Create client/src/viewer/sliceExtractor.js — pure functions:
   - `extractAxialSlice(volume, z, dimX, dimY)` — returns `volume.subarray(z * dimX * dimY, (z + 1) * dimX * dimY)` (zero-copy)
   - `extractCoronalSlice(volume, y, dimX, dimY, dimZ)` — allocates Float32Array(dimX * dimZ), loops z then x: `slice[z * dimX + x] = volume[x + y * dimX + z * dimX * dimY]`
   - `extractSagittalSlice(volume, x, dimX, dimY, dimZ)` — allocates Float32Array(dimY * dimZ), loops z then y: `slice[z * dimY + y] = volume[x + y * dimX + z * dimX * dimY]`

5. Create client/src/viewer/windowLevel.js:
   - `applyWindowLevel(sliceData, rgba, windowCenter, windowWidth)` — takes Float32Array slice, writes to Uint8ClampedArray rgba buffer. For each pixel: `scaled = ((raw - (center - width/2)) / width) * 255`, clamp 0-255, write R=G=B=val, A=255. This is the per-pixel float32 path (no LUT needed for float32 per RESEARCH.md recommendation).
   - Export the function.
