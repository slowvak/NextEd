   a. **Click+drag crosshair (per D-04, D-05):** On mousedown (without Ctrl/Meta), start crosshair tracking. This provides fluid click+drag crosshair navigation matching ITK-SNAP.
   b. **Ctrl+drag W/L (per D-05, WLVL-02):** On mousedown with Ctrl or Meta key, start W/L drag. Track lastX/lastY. On mousemove: dx = clientX - lastX, dy = clientY - lastY. Sensitivity = state.windowWidth / 300. New width = Math.max(1, state.windowWidth + dx * sensitivity). New center = state.windowCenter + dy * sensitivity (down=darker=higher center, per RESEARCH Pattern 6 note). Call state.setWindowLevel(newCenter, newWidth). Update lastX/lastY. On mouseup, stop. Update the slice axis for this panel: axial -> cursor[2], coronal -> cursor[1], sagittal -> cursor[0]. Call state.setCursor() which triggers crosshair sync in all panels (per D-02).
