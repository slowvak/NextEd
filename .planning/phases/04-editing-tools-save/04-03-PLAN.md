# Plan 04-03: Undo History & Hotkeys

<objective>
Implement a sparse-diff undo system for brush strokes to cap memory usage, along with keyboard shortcuts for streamlined editing.
</objective>

<requirements>
- EDIT-09: Undo/Redo history (local) capped at 3 actions
- KEYS-01: Basic hotkeys (Ctrl+Z)
</requirements>

## Tasks

<task>
<action>
Implement `UndoStack` map logic in `client/src/viewer/ViewerState.js`.
- Add `this.undoStack = [];` to constructor.
- Add `pushUndo(diff)`: receives an object `{ indices: Int32Array | Array, oldValues: Uint8Array | Array }`. Pushes to stack. If `this.undoStack.length > 3`, shift the oldest element to cap at length 3. Call `.notify()`.
- Add `undo()`: if `this.undoStack.length === 0` return; Pop the last diff. Loop through its elements and restore: `this.segVolume[diff.indices[i]] = diff.oldValues[i]`. Call `.notify()`.
</action>

<task>
<action>
Record stroke diffs in `client/src/viewer/ViewerPanel.js`.
- On `mousedown` (if tools are active): create `this._currentDiff = { indices: [], oldValues: [], seen: new Set() }`.
- Inside `_applyBrush`: Before updating `state.segVolume[idx]`, check `!this._currentDiff.seen.has(idx)`. If not seen, push `idx` to `indices`, push `state.segVolume[idx]` to `oldValues`, and `seen.add(idx)`.
- On `mouseup` (if painting): delete `seen`, and pass the remaining object to `this.state.pushUndo(this._currentDiff)`.
</action>
