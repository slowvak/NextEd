<task>
<action>
Implement `_applyBrush(e)` geometric logic in `ViewerPanel.js`.
- Use `canvasToVoxel()` to translate `(e.clientX, e.clientY)` into a central target voxel `[cx, cy, cz]`.
- Identify the plane dimensions: If axis is `axial`, X and Y vary while Z is depth (`cz`). If `coronal`, X and Z vary while Y is depth. If `sagittal`, Y and Z vary while X is depth.
- Loop over the depth range: `depth` from `[targetDepth - state.multiSlice]` to `[targetDepth + state.multiSlice]`. Clamp depth to `[0, dims[depthAxis] - 1]`.
- Loop over the plane grid: for `u` in `[cx - radius]` to `[cx + radius]`, and `v` in `[cy - radius]` to `[cy + radius]`.
- Compute distance `(u - cx)^2 + (v - cy)^2 <= R^2` (where `R = state.brushRadius`). If accurate, determine the 1D physical index `idx = z * (dimX * dimY) + y * dimX + x` based on `u`, `v`, and `depth`.
- If the index is valid: check constraint bounds. Retrieve original float volume intensity: `val = this.volume[idx]`. If `val >= state.paintConstraintMin && val <= state.paintConstraintMax`, proceed.
- If tool is `paint`, assign `state.segVolume[idx] = state.activeLabel`. If `erase`, assign `state.segVolume[idx] = 0`.
- Once loops complete, call `this.state.notify()` to re-render all slices using the updated buffer.
</action>
