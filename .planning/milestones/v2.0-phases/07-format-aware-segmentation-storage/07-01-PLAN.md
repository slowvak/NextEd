   **`remap_labels(seg_array: np.ndarray, label_names: dict[int, str]) -> tuple[np.ndarray, list[tuple[int, str]]]`**

   - Sort them, enumerate starting at 1
   - Build remapped array where old_val -> new_num
   - Return (remapped_array, [(seg_number, label_name), ...])

   **`_sort_dicom_datasets(datasets: list[pydicom.Dataset]) -> list[pydicom.Dataset]`**
   - Extract orientation from first dataset's ImageOrientationPatient
   - Compute slice_normal = cross(row_cosine, col_cosine)
   - Sort by dot(ImagePositionPatient, slice_normal)

   **`_ras_seg_to_dicom_frames(seg_zyx: np.ndarray, canonical_affine: np.ndarray, source_datasets: list[pydicom.Dataset]) -> np.ndarray`**
   - Input: seg in client shape (Z, Y, X), the canonical RAS+ affine, sorted source DICOM datasets

   - Step 2: Rebuild the original LPS affine from source DICOM geometry using `_build_affine` from dicom_loader (import it)
   - Step 3: Compute the RAS affine that would exist BEFORE as_closest_canonical: `ras_pre_canonical = np.diag([-1,-1,1,1]) @ lps_affine`
   - Step 4: Use `nibabel.io_orientation(ras_pre_canonical)` to get the orientation transform that `as_closest_canonical` applied
   - Step 5: Compute `ornt_canonical = nib.orientations.axcodes2ornt(('R','A','S'))`
   - Step 6: Get inverse transform: `inv_transform = nib.orientations.ornt_transform(ornt_canonical, ornt_orig)`
   - Step 7: Apply `nib.orientations.apply_orientation(seg_xyz, inv_transform)` to get back to original (rows, cols, slices) layout
   - Step 8: Transpose/reshape to (n_slices, rows, cols) for highdicom frames
   - Return the frames array as uint8

   **`build_dicom_seg(seg_zyx: np.ndarray, canonical_affine: np.ndarray, dicom_file_paths: list[str], label_names: dict[int, str], filename: str) -> tuple[pydicom.Dataset, list[tuple[int, str]]]`**

1. Create `server/watcher/suppress.py` with class `WatcherSuppressList`:
   ```python
   import threading
   import time

   class WatcherSuppressList:
       """Thread-safe set of paths the watcher should ignore temporarily.

       Used by the save endpoint to prevent the watcher from re-detecting
       DICOM-SEG files that the server itself just wrote.
       """

       def __init__(self, ttl: float = 5.0):
           self._paths: dict[str, float] = {}  # path -> expiry timestamp
           self._lock = threading.Lock()
           self._ttl = ttl

       def add(self, path: str) -> None:
           """Add a path to suppress. It will auto-expire after TTL seconds."""
           with self._lock:
               self._paths[path] = time.monotonic() + self._ttl

       def should_suppress(self, path: str) -> bool:
           """Check if a path should be suppressed. Cleans up expired entries."""
           with self._lock:
               expiry = self._paths.get(path)
               if expiry is None:
                   return False
               if time.monotonic() > expiry:
                   del self._paths[path]
                   return False
               return True

       def remove(self, path: str) -> None:
           """Explicitly remove a path from the suppress list."""
           with self._lock:
               self._paths.pop(path, None)
   ```
