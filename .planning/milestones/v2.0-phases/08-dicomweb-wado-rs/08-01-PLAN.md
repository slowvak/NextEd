  <action>
**Create `server/api/wado.py`** with the following structure:

1. **Router**: `APIRouter(prefix="/api/v1/wado-rs", tags=["wado-rs"])`  (per D-03)

2. **Helper `_resolve_series_files(study_uid: str, series_uid: str) -> list[str]`**:
   - Import `_metadata_registry` and `_path_registry` from `server.api.volumes`
   - Iterate `_metadata_registry` items
   - For each entry, check `meta.format == "dicom_series"` AND `meta.study_instance_uid == study_uid` AND `meta.series_instance_uid == series_uid`
   - When matched, get `(path_str, fmt)` from `_path_registry[vol_id]`
   - Return `json.loads(path_str)` (the list of file paths)
   - If no match found, return empty list `[]`

3. **Multipart generator `_multipart_generator(file_paths: list[str], boundary: str)`**:
   - For each file path, yield: `b"--" + boundary.encode() + b"\r\n"`, then `b"Content-Type: application/dicom\r\n"`, then `b"\r\n"` (end of part headers), then file content in 65536-byte chunks, then `b"\r\n"`
   - After all files, yield closing boundary: `b"--" + boundary.encode() + b"--\r\n"`

4. **Retrieve endpoint** (per D-01, D-04, D-05, D-06, D-10):
   ```
   GET /studies/{study_uid}/series/{series_uid}
   ```
   - Call `_resolve_series_files(study_uid, series_uid)`
   - If empty list, raise `HTTPException(status_code=404, detail="Series not found")`
   - Check all files exist with `Path(f).exists()` -- if any missing, raise `HTTPException(status_code=404, detail=f"DICOM file missing from disk: {f}")`
   - Generate boundary with `uuid.uuid4().hex`
   - Return `StreamingResponse(_multipart_generator(files, boundary), headers={"Content-Type": f'multipart/related; type="application/dicom"; boundary={boundary}'})`
   - IMPORTANT: Pass Content-Type via `headers` dict, NOT via `media_type` parameter (per Pitfall 1 in research -- Starlette may normalize media_type)

5. **Metadata endpoint** (per D-01, D-07, D-08, D-09):
   ```
   GET /studies/{study_uid}/series/{series_uid}/metadata
   ```
   - Call `_resolve_series_files(study_uid, series_uid)`
   - If empty, raise HTTPException 404
   - For each file path:
     - Check exists, raise 404 if not (per D-10)
     - `ds = pydicom.dcmread(fpath, stop_before_pixels=True)`
     - Extract `sop_uid = str(getattr(ds, "SOPInstanceUID", "unknown"))`
     - Create bulk data handler using default-argument binding to avoid closure bug (Pitfall 3):
       ```python
       def bulk_handler(elem, _study=study_uid, _series=series_uid, _sop=sop_uid):
           return f"/api/v1/wado-rs/studies/{_study}/series/{_series}/instances/{_sop}/bulk/{elem.tag:08X}"
       ```
     - `json_dict = ds.to_json_dict(bulk_data_element_handler=bulk_handler)`
     - Manually inject PixelData BulkDataURI since stop_before_pixels=True omits it (per Open Question 1 in research):
       ```python
       json_dict["7FE00010"] = {
           "vr": "OW",
           "BulkDataURI": f"/api/v1/wado-rs/studies/{study_uid}/series/{series_uid}/instances/{sop_uid}/bulk/7FE00010"
       }
       ```
     - Append json_dict to metadata_list
   - Return JSONResponse with `media_type="application/dicom+json"` and content=metadata_list
